Student Progression Analysis Program
This Python program allows users to input marks for students and analyze their progression outcomes based on predefined criteria. The program generates a histogram visualizing the distribution of student outcomes.

Features
Input pass, defer, and fail credits for each student.
Determine the progression outcome for each student based on the credit inputs.
Display the results in a histogram showing the distribution of progression outcomes.
Save the student records to a text file (data.txt).
Requirements
Python 3.x
graphics.py library (used for graphical display)
Usage
InstallationEnsure you have Python installed on your system. If not, download and install it from python.org.
SetupDownload or clone the repository to your local machine.
Run the ProgramOpen a terminal or command prompt and navigate to the directory containing the program files.Run the program using the following command:
Copy code
python progression_analysis.py
Input MarksFollow the prompts to input the pass, defer, and fail credits for each student.
View Results
Once you have finished inputting data for all students, type 'q' when prompted to view the histogram and quit.
The program will display the histogram showing the distribution of progression outcomes.
Close the window to exit the program.
